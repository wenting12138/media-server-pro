WebRTC Test Console

One page for both playback and browser publish. Playback calls /webrtc/play and /webrtc/stop. Publish calls /webrtc/publish and /webrtc/unpublish. The current server publish path supports H264 video, with browser audio negotiated as Opus.

Publish

Capture local camera + microphone and push them into /webrtc/publish. This page will prefer H264 during video codec negotiation while browser audio stays on Opus.

Notes
